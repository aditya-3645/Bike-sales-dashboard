## 📖 Overview

This project presents an **interactive Bike Sales Analytics Dashboard** built entirely in **Microsoft Excel**. It transforms raw bike sales data into meaningful business insights using pivot tables, charts, slicers, and dynamic visualizations.

The dashboard enables users to explore sales performance, customer trends, and resale value across different brands, cities, and seller types, helping stakeholders make informed business decisions.

---

## 🎯 Problem Statement

Bike dealerships and businesses often struggle to derive actionable insights from large volumes of sales data. This project addresses that challenge by transforming raw transactional data into an intuitive dashboard that enables quick analysis of:

- 📈 Sales performance
- 🏍️ Brand-wise demand
- 🌍 City-wise sales distribution
- 👤 Seller type analysis
- 💰 Pricing & resale trends
